AStar-Rust
====================

## What is it?

An A* implementation in Rust, including core building blocks for making a pathfinding algorithm. It implements unidirectional A* with three different heuristics.

## Why is it?

I just do this when learning new programming languages.

See [here](https://github.com/TheBizzle/PathFinding) and [here](https://github.com/TheBizzle/AStar-Haskell) and [here](https://github.com/TheBizzle/AStar-Golang) for similar examples.
